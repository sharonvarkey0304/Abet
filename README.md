# Abet - Your University Companion App

**Abet** is a cross-platform mobile application built with **Flutter**, designed to simplify the university experience by providing students with essential tools and information in one place.



## Core Features

* **University Updates:** Real-time notifications and feed for the latest campus news, events, and academic announcements.
* **Academic Notes:** A centralized repository where students can upload, share, and access lecture notes and study materials.
* **Codey (AI Chatbot):** An intelligent, built-in assistant named **Codey** to help students with academic queries, navigation, and app support.
* **Book Marketplace:** An integrated platform for students to buy and sell second-hand textbooks directly with their peers on campus.

## App Architecture



* **Frontend:** Developed using Flutter and Dart for a smooth, high-performance UI on both Android and iOS.
* **Backend Integration:** (e.g., Firebase for real-time updates and database management).
* **Chatbot Logic:** Powered by (e.g., OpenAI API or Dialogflow) to drive "Codey's" intelligence.

## Project Structure

* `lib/screens/updates/`: Logic for the university news feed.
* `lib/screens/notes/`: File management and sharing system for notes.
* `lib/screens/chatbot/`: Integration and UI for Codey.
* `lib/screens/marketplace/`: Peer-to-peer book selling interface.

## Getting Started

1.  **Clone the Repo:**
    ```bash
    git clone [https://github.com/sharonvarkey0304/abet-university-app.git](https://github.com/sharonvarkey0304/abet-university-app.git)
    ```
2.  **Install Dependencies:**
    ```bash
    flutter pub get
    ```
3.  **Run the App:**
    ```bash
    flutter run
    ```
